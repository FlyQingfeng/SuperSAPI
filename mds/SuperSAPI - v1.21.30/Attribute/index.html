Attribute
以下是Attribute​类的成员变量和成员函数的详细说明，以表格形式列出：
成员变量
成员变量描述
​private properties: { [key: string]: any }​一个私有变量，存储属性的键值对。
​_obj: any​保存构造函数传入的对象引用，该对象必须具有getDynamicProperty​和setDynamicProperty​方法。
成员函数
成员函数描述
​constructor(obj: any)​构造函数，初始化属性对象，并从传入对象的动态属性中加载属性数据。
​private save()​将属性数据保存到传入对象的动态属性中。
​init(key: string, value: any)​初始化属性，只有当该属性尚未存在时才会设置，避免覆盖已有属性。
​set(key: string, value: any): void​设置属性值，如果属性已存在则更新它。
​get(key: string): any​获取指定键的属性值。
​delete(key: string): void​删除指定键的属性。
​has(key: string): boolean​检查指定键的属性是否存在。
​keys(): string[]​获取所有属性的键数组。
​values(): any[]​获取所有属性的值数组。
​entries(): [string, any][]​获取所有属性的键值对数组。
​clear(): void​清空所有属性。
详细描述
​Attribute​类用于管理一个对象的动态属性。它提供了一组方法来设置、获取、删除和查询属性。属性数据被存储在一个私有的properties​对象中，并且可以通过save​方法保存到构造函数传入的对象的动态属性中。
constructor​
构造函数检查传入对象是否具有getDynamicProperty​和setDynamicProperty​方法，如果不具备则抛出异常。然后，它尝试从传入对象的动态属性中加载属性数据。
private save​
私有方法，将properties​对象的数据保存到传入对象的动态属性中。
init​
初始化属性，只有当该属性尚未存在时才会设置，避免覆盖已有属性。
set​
设置属性值，如果属性已存在则更新它，并调用save​方法保存更改。
get​
获取指定键的属性值。
delete​
删除指定键的属性，并调用save​方法保存更改。
has​
检查指定键的属性是否存在。
keys​
获取所有属性的键数组。
values​
获取所有属性的值数组。
entries​
获取所有属性的键值对数组。
clear​
清空所有属性，并调用save​方法保存更改。
​Attribute​类提供了一个灵活的属性管理机制，使得可以方便地为对象添加、修改和查询动态属性。通过这个类，可以在游戏中创建具有动态属性的自定义对象，例如实体、玩家或物品。
