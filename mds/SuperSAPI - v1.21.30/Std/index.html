以下是一组实用工具函数的详细说明，这些函数用于类型转换、功能检测、枚举处理、JSON序列化和反序列化，包括处理循环引用的情况：
函数列表
1.
cast<T>(obj: any): T
2.
hasFun(obj: any, funcName: string): boolean
3.
enumKeyToString(enumObj: any, enumValue: any): string
4.
toJSON(value: any): string
5.
fromJSON(value: string): any
详细描述
1. cast<T>(obj: any): T
描述：这是一个类型转换函数，用于将任意类型的对象obj​转换为指定的类型T​。这在TypeScript中特别有用，用于强制类型转换。
参数：
​obj​: any​ - 要转换的对象。
返回：T​ - 转换后的对象。
2. hasFun(obj: any, funcName: string): boolean
描述：检查对象obj​是否具有名为funcName​的方法。
参数：
​obj​: any​ - 要检查的对象。
​funcName​: string​ - 方法的名称。
返回：boolean​ - 如果对象具有该方法，则返回true​；否则返回false​。
3. enumKeyToString(enumObj: any, enumValue: any): string
描述：从枚举对象enumObj​中找到与enumValue​值对应的键名。
参数：
​enumObj​: any​ - 枚举对象。
​enumValue​: any​ - 枚举值。
返回：string​ - 与给定值对应的键名。如果未找到，则返回空字符串。
4. toJSON(value: any): string
描述：将对象value​序列化为JSON字符串，同时处理循环引用的问题。
参数：
​value​: any​ - 要序列化的对象。
返回：string​ - 序列化后的JSON字符串。
5. fromJSON(value: string): any
描述：将JSON字符串value​反序列化为对象，同时恢复循环引用。
参数：
​value​: string​ - 要反序列化的JSON字符串。
返回：any​ - 反序列化后的对象。
使用示例
typescript
// 使用 cast 函数 const player = cast<SuperPlayer>(entity); // 使用 hasFun 函数检查对象是否有特定方法 if (hasFun(player, 'attack')) { player.attack(); } // 使用 enumKeyToString 获取枚举键名 const key = enumKeyToString(SomeEnum, someEnumValue); // 使用 toJSON 和 fromJSON 处理循环引用 const obj = { self: null }; obj.self = obj; const json = toJSON(obj); // 处理循环引用 const parsedObj = fromJSON(json); // 恢复循环引用
